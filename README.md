# create-jue

Create a new Nuxt project with TailwindCSS or @nuxt/ui.

## Installation

```bash
# Using bun (recommended)
bun create jue@nuxt my-app

# Using npm
npm create jue@nuxt my-app

# Install in current directory (must be empty)
bun create jue@nuxt .
```

## UI Framework Options

| Option          | Description                            |
| --------------- | -------------------------------------- |
| **TailwindCSS** | Utility-first CSS framework (default)  |
| **@nuxt/ui**    | Nuxt UI library with built-in Tailwind |

## Available Modules

- **@nuxt/icon** - 200,000+ icons
- **@nuxtjs/color-mode** - Dark mode support
- **@nuxt/image** - Image optimization
- **@pinia/nuxt** - State management
- **@vueuse/nuxt** - Vue composition utilities
- **@nuxtjs/i18n** - Internationalization
- **@nuxt/content** - File-based CMS
- **@nuxtjs/supabase** - Supabase integration

## Publishing

```bash
# Publish with nuxt tag
npm publish --tag nuxt
```
